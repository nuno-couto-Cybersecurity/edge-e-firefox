Browser Startup Automation Script

Description

Batch script created to automate the startup of daily work tools by opening multiple web applications and communication platforms with a single click.

Features

Automatically launches multiple web-based applications.
Opens Microsoft Teams and Outlook.
Uses small delays between launches to improve browser stability and loading performance.
Reduces repetitive manual actions at the beginning of each work shift.

Technologies

Windows Batch (.bat)
PowerShell
Microsoft Edge
Mozilla Firefox

Purpose

Designed to improve productivity by automating the opening of frequently used applications and reducing setup time for daily operations.
